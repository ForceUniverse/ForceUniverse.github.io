Force

Dart Force is a Realtime web framework for Dart. We will make it easy for you to create realtime applications with it, like a chat, interactive dashboard, multiplayer games, ... It has a lot of other component inside him, Force MVC, Cargo, Wired, ...

  • Socket.io like communication interface
  • Clientside DB api, with serverside validation rules
  • Easy to use with Polymer and Angular.dart

Force MVC

A serverside mvc framework with template (mustache by default) into him. Easy for developers how are familiar with java spring mvc.

  • Mustache templates
  • Metadata, annotations, @Controller, @RequestMapping, ...
  • Security
  • Restfull api's with json
